# Qiita collector

Qiita collector display new article in [Qiita](http://qiita.com/)

# Usage

```
$ pip install -r requirements.txt
$ python3 qiita-collector.py
```

Demo: [https://qiita-collector.herokuapp.com/](https://qiita-collector.herokuapp.com/)
