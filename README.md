# Qiita collector

Qiita collector display new article in [Qiita](http://qiita.com/)

# Usage

```
$ pip install -r requirements.txt
$ python3 qiita-collector.py
```

Demo: [https://stormy-tor-35753.herokuapp.com/](https://stormy-tor-35753.herokuapp.com/)
