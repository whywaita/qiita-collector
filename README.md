# Qiita collector

Qiita collector display new article in [Qiita](http://qiita.com/)

# Usage

```
$ pip install -r requirements.txt
$ python3 manage.py
```

Demo: [https://qiita-collector.herokuapp.com/](https://qiita-collector.herokuapp.com/) (admin / password)
